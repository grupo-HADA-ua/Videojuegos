Tienda de Videojuegos
=====================

## Colaborar en el proyecto

Cómo bajarse el proyecto:
	
	git clone https://github.com/grupo-HADA-ua/Videojuegos.git

Cómo subir los cambios realizados:

Primero hay que añadir los cambios realizados, por ejemplo, si queremos añadir todo lo modificado:
	
	git add --all

Luego hay que hacer un commit:
	
	git commit -m 'mensaje descriptivo del commit'

Y finalmente hay que subir los cambios al repositorio remoto, por ejemplo, si queremos subir
los cambios al repositorio origin rama master:
	
	git push -u origin master
